# HTML Portfolio Project

This folder contains the `index.html` page for a simple HTML portfolio project.
links:https://dhruv1176.github.io/Portfolio/

## Overview

The page is a personal portfolio-style website featuring:

- A title and introduction for a web developer profile
- A small film recommendation section with the top 3 favorite movies
- A birthday invite section with an image, list of items to bring, and a link to a Google Maps location
- A link to a birthday invite details page at `./public/birthday-invite.html`

## Files

- `index.html` - main portfolio page
- `public/birthday-invite.html` - linked birthday invite details page

## How to View

Open `index.html` in a web browser to preview the page.

## Notes

- The page currently uses plain HTML only
- The image is loaded from an external URL
- The movie section includes an external IMDB link
